# Replit Prompt: Dr Teal's x Aaron Donald Project Details Page

## What I Need

Build a single project details page for a video production company called Family Drama. This is a project details page for a commercial campaign we produced for Dr Teal's featuring Aaron Donald. The page sits underneath a primary video player (which I'll add later) and showcases the project through a mix of written content and behind the scenes photography.

## Design References

This page should follow the layout patterns used by Tool of North America on their project detail pages. Here are reference URLs for structure and visual tone:

- https://toolofna.com/work/featured/verizon-verizon-5g-stadium/
- https://toolofna.com/work/featured/land-rover-defender-who-says-cars-cant-dream/
- https://toolofna.com/work/featured/milkpep-every-womans-marathon/

Key design patterns from these references:
- Clean, dark or neutral background
- Large section headers
- Body text in a readable column width (not full bleed)
- Images break out wider than the text column, some go full bleed, some sit in 2-up grids
- Smooth scroll, generous whitespace between sections
- Sections flow vertically with alternating text and media blocks
- No sidebar, no complex navigation, just a clean vertical scroll

## Design Direction

- Background: dark (near black or very dark gray)
- Text: white or light gray for body, slightly larger/bolder for section headers
- Typography: clean sans-serif (Inter, Helvetica Neue, or similar)
- Image treatment: no borders, no rounded corners, images should feel cinematic
- Layout: centered content column for text (~700-800px max), images can go wider (~1100-1200px) or full bleed
- Spacing: generous padding between sections (80-120px)
- Mobile responsive
